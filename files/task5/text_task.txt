Опять проекты

ограничение по времени на тест: 1 секунда
ограничение по памяти на тест: 256 мегабайт
ввод: стандартный ввод
вывод: стандартный вывод

Задача создать новый проект "Супер-пупер компьютер 2000". Этот проект состоит из n частей. Каждая часть характеризуется временем выполнения и частями, которые должны быть уже готовы, чтобы её выполнить. То есть некоторые части необходимо делать только после того, как будут готовы другие части.

Так как Эдуард очень хороший руководитель, то все части проекта могут быть выполнены параллельно.

Помогите Эдуарду понять, за какое минимальное время проект полностью завершится (будут выполнены все части).

Входные данные
Первая строка входного файла содержит число n (1 ≤ n ≤ 10^5) — количество частей проекта.

Вторая строка содержит n натуральных чисел p1, p2, ..., pn, определяющих время выполнения каждой части в часах. Время для выполнения каждой части не превосходит 10^6 часов.

Каждая из последующих n строк входного файла описывает требования для начала выполнения части. Здесь i-я строка содержит список частей проекта Ki (1 ≤ Ki ≤ n), которые требуется для выполнения части с номером i, а также их номера. В i-й строке не повторяются номера частей. Количество всех частей Ki не превосходит 10^5.

Известно, что не существует циклических зависимостей в выполнении частей проекта и проект всегда можно выполнить.

Выходные данные
Выведите минимальное время в часах, необходимое для скорейшего выполнения проекта.
